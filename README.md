# À Table — Idées repas & courses

## Installation
```bash
npm run install:all
npm run dev
```

PC : http://localhost:5173
Mobile : utiliser l’adresse Network affichée par Vite, par exemple http://192.168.1.xx:5173

## Version
Page d’accueil avec logo, recherche de plats, connexion/nouveau utilisateur, top 5 recettes, catalogue API TheMealDB, filtres allergies, fiche étape par étape, panier/liste de courses, export TXT/CSV/impression.

Note : TheMealDB fournit surtout des recettes anglaises. L’application traduit les mots courants et conserve une base française locale. Pour publication commerciale, prévoir une API premium ou une base française propriétaire.
